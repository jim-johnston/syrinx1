How to Contribute a Sound

Visiki OT relies on zero-dependency, procedural Web Audio math. To build and submit a new alarm or UI sound:

  1. Navigate to WebAudio Designer.
  2. Drag and drop nodes (Oscillators, BiquadFilters, Gains) to design your soundscape.
  3. Click Export JS and copy the generated vanilla JavaScript.
  4. Wrap the code in a named function (e.g., playMyAlarm()) and add a new .sound-row to index.html.
  5. Submit a Pull Request to the syrinx1 GitHub Repository.

NPM & SCADA Integration

Syrinx1 provides stateless audio snippets. For looping alarms in React, Angular, or standard HTML5 HMIs, you must wrap our functions in a state-aware interval loop.
